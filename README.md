### GOAPPLE - Утилита для быстрой установки I3-wm.

### SETUP

```text
git clone https://github.com/Avdushin/GOAPPLE-CLI
cd GOAPPLE-CLI
./goapple
# or start Russian language
./goapple -ru
```
